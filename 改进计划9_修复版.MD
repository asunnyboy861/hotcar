# HotCar 改进计划 9 - 数据流优化与用户体验增强 (修复版)

> **版本**: 1.1 (修复版)  
> **日期**: 2026-03-13  
> **目标**: 优化设置数据持久化、增强错误处理、完善空状态引导  
> **影响范围**: Settings, Weather, Statistics, Onboarding 模块

---

## 📋 执行摘要

基于对 HotCar 软件用户使用流程和数据流的深度分析，本改进计划针对以下核心问题进行优化：

1. **设置数据持久化问题**: 温度单位自动检测会覆盖用户手动设置
2. **设置保存性能问题**: 每次属性变更都同步写入 UserDefaults，缺少防抖
3. **统计数据一致性**: sessions 和 statistics 存在数据不一致风险
4. **错误处理不足**: 天气 API 失败、位置权限被拒绝等场景缺少降级方案
5. **空状态缺失**: 统计数据为空时缺少引导性提示
6. **数据导出功能**: 有方法但缺少 UI 入口

**修复说明**: 本修复版解决了原改进计划中与现有代码架构的冲突问题，确保改进方案与现有 SettingsService、LocationService 等模块兼容。

## 🎯 第一部分：需求描述

### 1.1 设置数据持久化优化

**问题描述**:
```swift
// AppSettings init 中
init() {
    let locale = Locale.current
    let regionCode = locale.region?.identifier ?? "US"
    let fahrenheitRegions = ["US", "LR", "MM"]
    self.temperatureUnit = fahrenheitRegions.contains(regionCode) ? .fahrenheit : .celsius
}
```

**问题影响**:
- 用户手动修改温度单位后，下次启动应用会被重置
- 用户体验差，设置无法持久保存

**需求**:
- 添加 `isTemperatureUnitOverridden` 标记，区分用户手动设置和自动检测
- 首次启动使用自动检测，后续优先使用用户保存的设置
- 用户手动修改后，标记为已覆盖，不再自动重置

**修复要点**: 
- 不在 `AppSettings.init()` 中加载 UserDefaults（避免与 `SettingsService` 重复）
- 由 `SettingsService` 控制是否使用地区检测
- `AppSettings` 只负责存储 `isTemperatureUnitOverridden` 标记

### 1.2 设置保存防抖优化

**问题描述**:
```swift
// SettingsViewModel 中
func saveSettings() {
    settingsService.updateTemperatureUnit(temperatureUnit)
    settingsService.updateDefaultTimerDuration(defaultTimerDuration)
    // 每次调用都同步写入 UserDefaults
}
```

**问题影响**:
- 快速切换设置（如连续点击 Stepper）会导致频繁写入 UserDefaults
- 可能影响性能

**需求**:
- 使用 Combine 的 `debounce` 操作符实现防抖
- 延迟 0.5 秒后保存，避免频繁写入
- 应用退出时立即保存，确保数据不丢失

**修复要点**:
- 需要添加 `import Combine`
- 使用 `PassthroughSubject` 触发保存
- 在 `deinit` 或应用生命周期事件中立即保存

### 1.3 统计数据一致性优化

**问题描述**:
```swift
// StatisticsService
@Published var sessions: [WarmUpSession] = []
@Published var statistics: WarmUpStatistics = WarmUpStatistics()
```

**问题影响**:
- `statistics` 是基于 `sessions` 计算的，但两者独立发布
- 直接修改 `sessions` 不会自动触发 `statistics` 更新
- 容易遗漏 `calculateStatistics()` 调用

**需求**:
- 使用 Combine 将 `statistics` 绑定到 `sessions`
- 任何 `sessions` 变更自动触发 `statistics` 重算
- 移除手动调用 `calculateStatistics()` 的需求

**修复要点**:
- 保持 `getStatistics(for:)` 方法不变，该方法用于特定车辆的统计
- `statistics` 属性始终为全局统计，通过 Combine 自动更新
- 特定车辆统计通过方法动态计算，不存储在属性中

### 1.4 天气服务错误处理增强

**问题描述**:
```swift
// WeatherService
catch {
    await MainActor.run {
```

**问题影响**:

**需求**:
- 添加错误状态枚举，区分网络错误、权限错误、服务器错误
- API 失败时使用缓存数据或默认值
- 添加位置权限被拒绝的引导提示
- 提供手动刷新按钮和重试机制

**修复要点**:
- 复用 `LocationService.authorizationStatus` 检查权限
- 添加天气数据缓存机制（独立于 LocationService 的缓存）
- 错误提示通过 `@Published` 暴露给 View

### 2.2 设置保存防抖优化方案

#### 2.2.1 SettingsViewModel 防抖实现

```swift
import Foundation
import UIKit
import Combine  // 新增导入

@MainActor
final class SettingsViewModel: ObservableObject {
    
    // MARK: - Published Properties
    
    @Published var temperatureUnit: AppSettings.TemperatureUnit = .celsius
    @Published var defaultTargetTemp: Double = 20.0
    @Published var defaultTimerDuration: Int = 15
    @Published var autoStartTimer: Bool = false
    @Published var showNotifications: Bool = true
    @Published var hapticFeedback: Bool = true
    @Published var notificationSound: AppSettings.NotificationSound = .default
    @Published var darkMode: AppSettings.DarkModeSetting = .system
    @Published var analyticsEnabled: Bool = true
    @Published var crashReportingEnabled: Bool = true
    
    // MARK: - Private Properties
    
    private let settingsService = SettingsService.shared
    private var cancellables = Set<AnyCancellable>()
    private let saveSubject = PassthroughSubject<Void, Never>()
    
    // MARK: - Initialization
    
    init() {
        loadSettings()
        setupDebounce()
    }
    
    deinit {
        // 立即保存，确保数据不丢失
        saveImmediately()
    }
    
    // MARK: - Setup
    
    private func setupDebounce() {
        saveSubject
            .debounce(for: .milliseconds(500), scheduler: RunLoop.main)
            .sink { [weak self] in
                self?.performSave()
            }
            .store(in: &cancellables)
    }
    
    // MARK: - Public Methods
    
    /// 触发保存（带防抖）
    /// 在属性 didSet 中调用
    func triggerSave() {
        saveSubject.send()
    }
    
    /// 立即保存（用于应用退出时）
    func saveImmediately() {
        // 取消未执行的防抖任务
        cancellables.removeAll()
        performSave()
    }
    
    private func performSave() {
        settingsService.updateTemperatureUnit(temperatureUnit)
        settingsService.updateDefaultTimerDuration(defaultTimerDuration)
        settingsService.updateAutoStartTimer(autoStartTimer)
        settingsService.updateNotifications(showNotifications)
        settingsService.updateHapticFeedback(hapticFeedback)
        settingsService.updateNotificationSound(notificationSound)
        settingsService.updateDarkMode(darkMode)
    }
    
    func loadSettings() {
        temperatureUnit = settingsService.settings.temperatureUnit
        defaultTargetTemp = settingsService.settings.defaultTargetTemp
        defaultTimerDuration = settingsService.settings.defaultTimerDuration
        autoStartTimer = settingsService.settings.autoStartTimer
        showNotifications = settingsService.settings.showNotifications
        hapticFeedback = settingsService.settings.hapticFeedback
        notificationSound = settingsService.settings.notificationSound
        darkMode = settingsService.settings.darkMode
        analyticsEnabled = settingsService.settings.analyticsEnabled
        crashReportingEnabled = settingsService.settings.crashReportingEnabled
    }
    
    // ... 其他方法保持不变 ...
}
```

```swift
import Foundation
import Combine  // 确保导入

@MainActor
final class StatisticsService: ObservableObject {
    
    // MARK: - Published Properties
    
    @Published var sessions: [WarmUpSession] = []
    
    /// 自动计算的统计数据，基于 sessions
    /// 通过 Combine 自动更新，无需手动调用 calculateStatistics()
    @Published private(set) var statistics: WarmUpStatistics = WarmUpStatistics()
    
    // MARK: - Private Properties
    
    private let storageKey = "com.hotcar.sessions"
    private let vehicleService = VehicleService.shared
    private var cancellables = Set<AnyCancellable>()
    
    // MARK: - Initialization
    
    private init() {
        loadSessionsSync()
        setupBindings()
    }
    
    // MARK: - Combine Bindings
    
    private func setupBindings() {
        $sessions
            .receive(on: RunLoop.main)
            .sink { [weak self] sessions in
                self?.calculateStatistics()
                self?.saveSessions()
            }
            .store(in: &cancellables)
    }
    
    // MARK: - Public Methods
    
    func addSession(_ session: WarmUpSession) async {
        sessions.append(session)
        // statistics 会自动更新，无需手动调用
    }
    
    func updateSession(_ session: WarmUpSession) async {
        if let index = sessions.firstIndex(where: { $0.id == session.id }) {
            sessions[index] = session
            // statistics 会自动更新
        }
    }
    
    func deleteSession(_ session: WarmUpSession) async {
        sessions.removeAll { $0.id == session.id }
        // statistics 会自动更新
    }
    
    /// 获取特定车辆的统计
    /// 注意：此方法动态计算，不依赖 @Published statistics
    func getStatistics(for vehicleId: String? = nil) -> WarmUpStatistics {
        guard let vehicleId = vehicleId else {
            return statistics  // 返回全局统计
        }
        
        let filteredSessions = sessions.filter { $0.vehicleId == vehicleId }
        return calculateStatistics(for: filteredSessions)
    }
    
    // MARK: - Private Methods
    
    private func calculateStatistics() {
        statistics = calculateStatistics(for: sessions)
    }
    
    private func calculateStatistics(for sessions: [WarmUpSession]) -> WarmUpStatistics {
        var stats = WarmUpStatistics()
        
        stats.totalSessions = sessions.count
        stats.totalMinutes = sessions.reduce(0) { $0 + $1.durationMinutes }
        stats.totalFuelUsed = sessions.reduce(0) { $0 + $1.fuelEstimate }
        stats.totalCost = sessions.reduce(0) { $0 + $1.costEstimate }
        
        if !sessions.isEmpty {
            stats.averageDuration = Double(stats.totalMinutes) / Double(stats.totalSessions)
            stats.averageTemperature = sessions.reduce(0) { $0 + $1.outsideTemperature } / Double(stats.totalSessions)
        }
        
        // ... 其他计算逻辑 ...
        
        return stats
    }
    
    // ... 其他方法 ...
}
```

**检验标准**:
- [ ] 添加/删除/修改 session 后 statistics 自动更新
- [ ] 无需手动调用 calculateStatistics()
- [ ] 统计数据始终与 sessions 保持一致
- [ ] getStatistics(for:) 方法仍然正常工作

```swift
// MARK: - Weather Error

enum WeatherError: Error, LocalizedError {
    case invalidURL
    case networkError(Error)
    case serverError(Int)
    case decodingError(Error)
    case locationPermissionDenied
    case locationUnavailable
    case noCachedData
    
    var errorDescription: String? {
        switch self {
        case .invalidURL:
            return "Invalid weather service URL"
        case .networkError:
            return "Network connection failed. Please check your internet connection."
        case .serverError(let code):
            return "Weather service unavailable (Error \(code)). Please try again later."
        case .decodingError:
            return "Failed to parse weather data"
        case .locationPermissionDenied:
            return "Location access is required for accurate weather data"
        case .locationUnavailable:
            return "Unable to determine your location"
        case .noCachedData:
            return "No cached weather data available"
        }
    }
    
    var recoverySuggestion: String? {
        switch self {
        case .locationPermissionDenied:
            return "Please enable location access in Settings > Privacy > Location Services"
        case .networkError, .serverError:
            return "Pull down to refresh or check your connection"
        case .locationUnavailable:
            return "Please check your location settings and try again"
        default:
            return nil
        }
    }
}
```

#### 2.4.2 WeatherService 错误处理增强

**文件**: `Features/WeatherIntegration/WeatherService.swift`

**修改内容**:
```swift
@MainActor
final class WeatherService: ObservableObject {
    
    // ... 现有属性 ...
    
    /// 当前错误状态
    @Published var currentError: WeatherError?
    
    /// 是否显示错误提示（用于触发 alert）
    @Published var showError: Bool = false
    
    /// 是否使用缓存/默认数据
    @Published var isUsingFallbackData: Bool = false
    
    /// 天气数据缓存
    private var cachedTemperature: Double?
    private let weatherCacheKey = "com.hotcar.weather.cache"
    private let weatherCacheTimestampKey = "com.hotcar.weather.cacheTimestamp"
    private let cacheValidDuration: TimeInterval = 3600 // 1 hour
    
    // MARK: - Public Methods
    
    func fetchCurrentTemperature() async {
        isLoading = true
        currentError = nil
        isUsingFallbackData = false
        
        // 复用 LocationService 的权限状态
        let authStatus = LocationService.shared.authorizationStatus
        guard authStatus == .authorizedWhenInUse || authStatus == .authorizedAlways else {
            if authStatus == .denied || authStatus == .restricted {
                handleError(.locationPermissionDenied)
            } else {
                handleError(.locationUnavailable)
            }
            useCachedDataOrDefault()
            return
        }
        
        // Get location from LocationService
        let location = LocationService.shared.getCurrentLocation()
        
        let urlString = "\(baseURL)?latitude=\(location.latitude)&longitude=\(location.longitude)&current_weather=true&temperature_unit=celsius"
        
        guard let url = URL(string: urlString) else {
            handleError(.invalidURL)
            useCachedDataOrDefault()
            return
        }
        
        do {
            let (data, response) = try await URLSession.shared.data(from: url)
            
            guard let httpResponse = response as? HTTPURLResponse else {
                handleError(.networkError(NSError(domain: "Invalid response", code: -1)))
                useCachedDataOrDefault()
                return
            }
            
            guard (200...299).contains(httpResponse.statusCode) else {
                handleError(.serverError(httpResponse.statusCode))
                useCachedDataOrDefault()
                return
            }
            
            let decoder = JSONDecoder()
            let weatherData = try decoder.decode(WeatherResponse.self, from: data)
            
            currentTemperature = weatherData.currentWeather.temperature
            cacheTemperature(weatherData.currentWeather.temperature)
            lastUpdateTime = Date()
            isLoading = false
            
        } catch let decodingError as DecodingError {
            handleError(.decodingError(decodingError))
            useCachedDataOrDefault()
        } catch {
            handleError(.networkError(error))
            useCachedDataOrDefault()
        }
    }
    
    // MARK: - Private Methods
    
    private func handleError(_ error: WeatherError) {
        currentError = error
        showError = true
        isLoading = false
        print("WeatherService Error: \(error.localizedDescription)")
    }
    
    private func cacheTemperature(_ temperature: Double) {
        UserDefaults.standard.set(temperature, forKey: weatherCacheKey)
        UserDefaults.standard.set(Date().timeIntervalSince1970, forKey: weatherCacheTimestampKey)
        cachedTemperature = temperature
    }
    
    private func useCachedDataOrDefault() {
        isUsingFallbackData = true
        
        // 尝试使用缓存数据
        if let cached = UserDefaults.standard.object(forKey: weatherCacheKey) as? Double,
           let timestamp = UserDefaults.standard.object(forKey: weatherCacheTimestampKey) as? TimeInterval {
            let cacheDate = Date(timeIntervalSince1970: timestamp)
            // 缓存 1 小时内有效
            if Date().timeIntervalSince(cacheDate) < cacheValidDuration {
                currentTemperature = cached
                return
            }
        }
        
        // 使用默认温度（0°C）
        currentTemperature = 0
        if currentError == nil {
            currentError = .noCachedData
        }
    }
    
    /// 清除错误状态（用于用户手动关闭错误提示后）
    func clearError() {
        currentError = nil
        showError = false
    }
}
```
